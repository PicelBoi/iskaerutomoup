Kaerutomo is down!

  • First, try this:
    • Does it respond with an error code along the lines of "801-1450-xxxx"? If you're on Android 8.1 or older, you may need to fix IdenTrust. See here for the fix.
    • Otherwise, follow these instructions if you're still encountering an error along the lines of "801-1450-xxxx". See here for the fix.
    • Does it respond with 801-1350-2407? Log into Google Play or Install Google Play Services/microG if on a custom ROM.
    • Did you recently just start it up? Follow these instructions.
    • Does it respond with 801-0011-0000? Check this out.
    • Still doesn't work? Go to the support channel on the Kaeru Community Discord and try to explain what steps you have taken to solve this.
    • If it's apparent Kaerutomo is actually down, contact me at @picelboi on Discord, email me at picelboi@picelboi.xyz, or make a pull request to this GitHub repository.
